# 📌 캡스톤 디자인 프로젝트 개발계획서

## 1. 프로젝트 개요
- **프로젝트 이름:** 서버 클라이언트 구조의 CD Measurer/Aligner 구현 
  (HBS Solution 기업 연계)
- **개발 기간:** 2025년 3월 ~ 2025년 6월
- **사용 기술:**
  - **백엔드:** C++(MFC)
  - **프론트엔드:** MFC GUI
  - **데이터베이스:** MySQL
  - **버전 관리:** Git, GitHub
  - **기타** OpenCV, TCP/IP 통신, Basler 등
