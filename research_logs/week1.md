# 📌 캡스톤 디자인 프로젝트 개발계획서

## 1. 프로젝트 개요
- **프로젝트명:** 스터디 모임 찾기 웹페이지
- **개발 기간:** 2025년 3월 ~ 6월
- **사용 기술:**
  - **백엔드:** Spring Boot
  - **프론트엔드:** Vue.js
  - **데이터베이스:** MySQL
  - **배포:** Docker

---
